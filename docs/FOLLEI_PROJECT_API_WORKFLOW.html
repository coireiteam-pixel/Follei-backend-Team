API எங்கே create ஆகிறது?

  1. app/routers/*.py-ல் decorator மூலம் endpoint எழுதப்படுகிறது.
  2. உதாரணம்: @router.get("").
  3. app/main.py-ல் app.include_router(...) மூலம் app-க்கு register ஆகிறது.
  4. API_PREFIX = "/api" சேர்ந்து final URL உருவாகிறது.

Schema ஏன்?

app/schemas/*.py request body மற்றும் response shape-ஐ define செய்கிறது. Missing required field அல்லது wrong datatype வந்தால் FastAPI automatic 422 return செய்யும்.

Lead import rules

  • Only .csv, UTF-8.
  • CSV id இருந்தால் preserve; இல்லையெனில் Lxxx generate.
  • tags JSON list; custom_fields and metadata JSON objects.
  • Same ID மீண்டும் upload செய்தால் update count-ல் வரும்.

Revenue import rules

  • ஒவ்வொரு row-க்கும் valid lead_id தேவை.
  • value required number.
  • probability 0 முதல் 1 வரை.
  • weighted_revenue = value × probability.
Import order: leads.csv first, revenue.csv second. இந்த data memory-ல் இருப்பதால் Uvicorn restart / reload ஆனதும் imported rows disappear ஆகும்.

12. Important current limitations

Lead persistence: Leads/revenue SQL models அல்லது DB tables-க்கு router migrate செய்யப்படவில்லை.
Reload behavior: --reload source change detect செய்தால் process restart; memory data clear.
ID schema mismatch: Existing PostgreSQL UUID schema மற்றும் current short-string ORM ஒன்றாக பயன்படுத்த முடியாது.
Auth coverage: DB modules tenant-authenticated; memory modules பெரும்பாலும் unauthenticated.
RAG search: Current visible search lexical term matching; full vector search endpoint அல்ல.
Document upload: Current documents create API file metadata-only; binary storage handling வேறு implementation தேவை.
